# libopus prebuilts

This directory contains Android shared-library builds of official libopus 1.6.1
for the ABIs used by WDCable Android Audio Link v2.

- Source: https://downloads.xiph.org/releases/opus/opus-1.6.1.tar.gz
- Release notes: https://opus-codec.org/release/stable/2026/01/14/libopus-1_6_1.html
- Source SHA256: `6ffcb593207be92584df15b32466ed64bbec99109f007c82205f0194572411a1`
- ABIs: `arm64-v8a`, `x86_64`
- Build type: CMake `Release`, `OPUS_BUILD_SHARED_LIBRARY=ON`

The upstream license text is copied in `OPUS_COPYING.txt`.
